Use the left-panel buttons to select scenes, control settings, etc. Selecting each option brings up the corresponding docs in the right panel. The button on the right allows you to return to the introduction text.

Left Stick - Strafe Left/Right
<ONLY_RIFT>Press Left Stick - Run

Right Stick - Smooth rotate
Start - Open GUI

<title>Keyboard + Mouse </title>
W,A,S,D - Move forward/backward + Strafe Left/Right
Left Shift- Run

Mouse X Axis - Smooth rotate
